# ChatIA - RAG Analysis Platform

Plataforma basada en RAG para análisis inteligente de documentos PDF.

## 🚀 Características
- **FastAPI Backend**: Procesamiento rápido y asíncrono.
- **Gradio Frontend**: Interfaz intuitiva y moderna.
- **MemoRAG**: Soporte para documentos extensos.
- **Bypass de Cloudflare**: Canal dedicado para archivos pesados.
- **Docker Ready**: Despliegue sencillo con un solo comando.
